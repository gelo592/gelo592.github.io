brain billy

site

a beta version of an online platform for teaching english through cognitive games. each game is based on a cognitive science idea for helping memory and retention targeting one of the key areas of language learning. the aim was to teach professionals in a dopamine stimulating environment which is essential to retaining knowledge.

postgresql db, flask app, angularjs frontend running through nginx on an aws ec2 instance.
