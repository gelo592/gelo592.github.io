Brain Billy

site defunct

A beta version of an online platform for teaching english through cognitive games. Each game is based on a cognitive science idea for helping memory and retention targeting one of the key areas of language learning. The aim was to teach professionals in a dopamine stimulating environment which is essential to retaining knowledge.

Postgresql DB, flask server, angularJS frontend running through nginx on an AWS ec2 instance.

Mail Billy (v1)

site

A speed translation service that returns small texts translated, corrected, or enhanced in under an hour. For larger text a time estimate is delivered in an hour. This service goes beyond verbatim translations by adding a native vocabulary and flair that will appeal to the target audience.

website using stripe and running on heroku

Integrative Med

site

website hosted through amazon s3

Collegiate Kicks

site
